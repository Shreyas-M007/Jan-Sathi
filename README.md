# Jan Sathi 🏙️🤖

> **Transforming public issue reporting with Autonomous AI Triage.**

Built during a 3-hour sprint for the **PROTOCOL BMSCE Beginner Hackathon**, Jan Sathi is a smart civic issue reporting platform. It directly tackles **Problem Statement 2: Public Issue Reporting and Resolution Tracker** by replacing manual government backlogs with instant, AI-driven sorting.

---

## 🚀 The Problem
Citizens frequently report civic issues like potholes, broken streetlights, and garbage overflow. However, these reports often sit in a massive, unorganized queue. There is a lack of transparency for the citizen and a critical lack of efficient triage for the authorities.

## 💡 Our Solution
Instead of a standard database that just stores complaints, Jan Sathi acts as an **Autonomous Triage Agent**. 

When a user uploads a photo of a hazard, the system uses the Gemini API to instantly:
1. **Classify the issue** (e.g., Deep Pothole, Fallen Power Line, Organic Waste).
2. **Score the severity** on a scale of 1-10 based on immediate public danger.
3. **Route the ticket** to the appropriate department dashboard.

This ensures that critical hazards automatically jump to the top of the city's queue, rather than waiting behind minor complaints.

## ✨ Key Features
* **Frictionless Citizen Reporting:** A mobile-first interface allowing users to snap a photo, tag their location, and submit a report in seconds.
* **Emergent AI Triage:** Zero-human-intervention sorting powered by advanced vision LLMs.
* **Smart Admin Dashboard:** A dedicated view for city officials where issues are automatically ranked by maximum urgency, saving hundreds of hours of manual sorting.
